Overview PROJETO ESTAÇÃO AR

Visão Geral do Projeto

Objetivo Principal

Integrar sensores ambientais externos como recursos nativos do sistema Android. Permitir que apps acessem dados de qualidade do ar através da API padrão SensorManager.

Integração & Conexão

  • Atual: Comunicação USB Serial (OTG) com protocolo JSON.

  • Futuro: Planejamento para canal Wi-Fi (TCP Sockets).

  • Exposição via SensorService para todo o sistema.
