Troubleshooting DESAFIOS & SOLUÇÕES

Dificuldades e Soluções (Parte 1)

Erro de Dependência (Soong)

Build falhava por módulos jsoncpp ausentes.

"sensors.airquality" depends on undefined module "libjsoncpp_static"

Ajuste no Android.bp

Configuração das shared_libs e headers.

shared_libs: ["libjsoncpp"], header_libs: ["libjsoncpp_headers"]

Include Não Encontrado

Header base 'V2_0/SubHal.h' invisível.

fatal error: 'V2_0/SubHal.h' file not found

Header Lib Específico

Inclusão da lib correta da interface Multi-HAL.

header_libs: [ "android.hardware.sensors@2.X-multihal.header" ]

Versão Rejeitada

Framework rejeitava a HAL por versão incorreta.

SubHal version was not 2.0

Definição Hexadecimal

Uso da constante exata exigida pelo HalProxy.

*version = SUB_HAL_2_0_VERSION; // 0x02000000
